# Pixeldrain Uploader

<img width="527" height="365" alt="Screenshot" src="https://github.com/user-attachments/assets/d1f01d4e-29eb-42a1-affb-062a2c820a7a" />

GNOME Shell extension to upload files to Pixeldrain from the top panel with quick upload and recent links.

## Notes
- The extension uses the Pixeldrain API and requires a user [API key](https://pixeldrain.com/user/api_keys) 
- Uploaded file links are copied to the clipboard automatically after a successful upload
- Recent uploads are stored locally in the extension settings

## Preferences
Open the extension preferences from GNOME Extensions and set your Pixeldrain API key.
